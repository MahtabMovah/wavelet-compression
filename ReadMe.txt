
Unzip the folder to where you want.
To run the code from command line, first compile with:

>> javac HomeworkThree.java

and then, you can run it to take in two parameters "pathToRGBVideo" and "n":

>> java wavelet-compression pathToRGBVideo n

where the parameter "n" is a string parameter used to show the decoding level and takes 0 to 9 and -1 for progressive decoding
